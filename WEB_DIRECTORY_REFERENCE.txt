1. OVERVIEW
============================================================

The web directory contains a Next.js web application for the Kerala Ayurvedh / Ayurvedic weight-loss product website.

It is currently a frontend-focused project. It has a branded landing page, static product sections, image-based hero slideshow, customer reviews, login/register screens, and a mock partner dashboard.

At the current stage, the project does not yet have a real backend, database, authentication system, product catalog API, cart logic, search logic, referral tracking, or commission calculation. The login, register, cart, search, and dashboard features are mostly visual/prototype features.

Main technologies used:

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Lucide React icons
- next/image for optimized images in some places
- next-themes installed for theme support
- Axios installed but currently unused
